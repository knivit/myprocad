Frequently Asked Questions

1. What is the program for ?

The program helps you easily draw house's plan using only a few graphical elements. If you wish, you can create a material's library, assign the materials to the elements and the program will calculate the needed amounts (in units and in cubic meters) and a price (if defined). Results can be stored in a file, printed or saved as PDF-file.

2. How to install it ?

Download its jar-file from our site MyProCAD and save it somewhere on your PC (on a desktop, for example).

3. How to remove it ?

Remove the downloaded jar-file.

4. Is it a virus ?

No. The program doesn't change Browser's settings, doesn't use a network, doesn't listen ports, doesn't spies for the keyboard, doesn't take screen shots. At any moment you can verify it by installing it into a virtual PC, sending it to a online virus-scanner etc. Its compiled jar-file isn't obfuscated and can be easily disassembled.

5. Can it damage anything on my PC ?

No. Below a full list of actions the program takes:
a) Creates .MyProCAD folder in user's home directory to keep its settings there (1-2 KB);
b) When saving a project, you do name a file to be saved;
c) When opening a project, you to select a file to be opened.
There are no actions with Windows registry, no shortcuts creation, it doesn't have an installer.

6. I have ideas how to improve it, what to do ?

Send an e-mail to support@myprocad.com

7. Why the program is absolutely FREE ?

It is absolutely FREE, because:
a) We think that the more free software, the better;
b) The program was developed using free software - Kubuntu, Subversion, Java, Maven, Intellij Idea;
c) There are about 30% of code from another free project Sweet Home 3D in it.
