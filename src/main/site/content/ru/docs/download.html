Журнал изменений

05-Сен-2014 Выпущена MyProCAD версия 1.0, поддерживающая все виды ОС

- Сделан сайт;
- Унифицировн интерфейс работы с папками, планами и записями проекта;
- Добавлена возможность добавления нескольских вкладок с записями;
- Добавлен расчет суммарной внешней длины выбранных стен методом сортировки по глубине;
- Сделана локализация на русский язык;
- Исправлены мелкие ошибки.

19-Май-2014 Добавлена поддержка Java 8

- Программа переписана с Java 7 на Java 8 с использованием потоков, лямбд и ссылок на функции;
- Сделана печать в PDF;
- Добавлены метки уровней.

22-Сен-2013 Первый закрытый выпуск - версия 0.1

- Добавлен редактор свойств;
- Убраны зависимости от закрытых библиотек;
- Добавлена печать сетки и линеек.
